# 📖 Sobre o Projeto

Este projeto consiste no desenvolvimento de um sistema web completo, simulando uma loja virtual da Dell.

A aplicação foi construída sem o uso de frameworks, com foco na organização de rotas, hierarquia de pastas e utilização correta de caminhos relativos entre arquivos HTML.

O sistema apresenta um site institucional navegável, permitindo ao usuário acessar informações da empresa, visualizar produtos e consultar páginas específicas de cada item.

# 🎯 Objetivo da Atividade
Desenvolver um sistema web estruturado

Aplicar conceitos de organização de diretórios

Utilizar corretamente caminhos relativos

Criar navegação entre páginas HTML

Seguir um template padrão em todas as páginas
